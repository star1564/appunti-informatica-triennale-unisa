#### ADDIZIONE
Se $a, b \in \mathbb{Z}$ è definita, allora la loro somma *$a+b \in \mathbb{Z}$*.
L'operazione di *Addizione* gode di alcune proprietà:
- [[040 Proprietà Operazioni in N0#^6107be|Associativa]];
- [[040 Proprietà Operazioni in N0#^cbb1f9|Commutativa]];
- L'**Elemento Neutro** rispetto all'addizione è 0:
$z+0=z,\quad$con $\forall a,b,c\in \mathbb{Z}$
- **Simmetrica**: $\forall a\in \mathbb{Z}, \exists -a\in \mathbb{Z}: -a+a=0=a+(-a)$
- $\exists:z+w=0,\quad$con $z, w\in \mathbb{Z}$;
- Se $a+b = 0 \implies a=-b \lor a=b=0,\quad$con $\forall a,b\in \mathbb{Z}$;

#### MOLTIPLICAZIONE
Se $a, b \in \mathbb{Z}$ è definita, allora la loro somma *$ab \in \mathbb{Z}$*.
L'operazione di *Moltiplicazione* gode di alcune proprietà:
- **Associativa**;
- **Distributiva**;
- Esiste un **Elemento neutro**;
- Se $a\times b = 1,\implies a = -b \lor a = b = -1\quad$con $\forall a,b\in \mathbb{Z}$.

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
