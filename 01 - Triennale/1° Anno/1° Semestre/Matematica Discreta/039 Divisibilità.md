>Con $x, y\in \mathbb{Z}$, si dice che $x$ **divide** $y$ e si scrive **$x|y$** se esiste *$k\in \mathbb{Z}$ tale che $y=x\cdot k$*. Ovvero:
> $$a|b \iff \exists k \in \mathbb{N_0}: b =  ak$$

Vale anche per gli altri [[004 Insiemi Numerici Infiniti|Insiemi Numerici]].

Si noti che: 
- con $x\neq 0$, *$x|y$* se e solo se *$resto(y,x)=0$*;
- *$0|y$* se e solo se *$y=0$*;
- *$z|0$*$, \forall z \in \mathbb{Z}$.

Se $\forall a\in \mathbb{N}$, $a$ possiede sempre almeno due divisori: *se stesso ($a$) ed 1*.

#### INSIEMI DI DIVISORI
Con $a \in \mathbb{Z}$, si pone *l'insieme dei divisori di $a$*:
$$D(a)=\{z\in \mathbb{Z}: z|a\}$$
Dove:
- $D(a)=D(-a)$;
- $D(1)=D(-1)=\{1,-1\}$;
- $D(0)=\mathbb{Z}$.

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

  // Use a regular expression to check if the string starts with the target number

	let res = []
