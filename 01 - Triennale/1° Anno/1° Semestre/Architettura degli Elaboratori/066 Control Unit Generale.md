>L'**Unità di controllo** genera tutti i 9 segnali di controllo necessari. Questi segnali vengono impostati dall'Unità di Controllo in *base al codice operativo dell'istruzione* (i primi 6 bit).

Ciascun segnale può essere 0, 1, X (indifferente), in funzione del valore del codice operativo.

#### PCSRC
Il segnale PCSrc deve valere 1 quando l'istruzione beq e l'uscita Zero della ALU sono vere, quindi si mette una porta AND prima del multiplexer.
![[Pasted image 20211211113454.png]]

#### PROGETTO
La CU è costruita tramite un [[023 PLA|PLA]]. Gli output di ALUop1 e di ALUop2 sono collegati alla [[065 Control Unit ALU|Unità di Controllo della ALU]].

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
