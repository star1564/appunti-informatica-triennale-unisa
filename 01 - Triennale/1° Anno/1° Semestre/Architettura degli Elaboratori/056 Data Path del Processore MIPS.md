Un processore MIPS a 32 bit può essere implementato in due modi:
- [[064 MIPS a Ciclo Singolo|A ciclo singolo]];
- [[067 Pipeline|Basata su pipeline]].

Le **unità funzionali** usate in entrambe le implementazioni sono:
- Il [[061 Program Counter|Program Counter (PC)]];
- La [[062 Memoria Istruzioni|Memoria Istruzioni]];
- Il [[058 Registri#BANCO DEI REGISTRI|Banco dei Registri]];
- La [[037 ALU|ALU]] (in cui si può ignorare Ainverter e la NOR);
- La [[063 Memoria Dati|Memoria Dati]].

Abbiamo bisogno delle **Unità di Controllo** per poter scegliere le operazioni da eseguire. Ci sono due Unità di Controllo:
- L'Unità di Controllo della ALU;
- L'Unità di Controllo Generale.

In generale, tutte e due le implementazioni sono divise nelle stesse **cinque fasi**: ^f11366
1. *Fetch (IF)*, il prelievo dell'istruzione dalla Memoria Istruzioni che contiene il programma;
2. *Dec (ID)*, la decodifica dell'istruzione e accesso ai Registri;
3. *Exe (EX)*, il calcolo del risultato attraverso l'ALU;
4. *Mem (MEM)*, l'accesso alla Memoria Dati, dove si prelevano o si depositano i dati necessari;
5. *Write Back (WB)*, la scrittura del risultato nel Banco dei Registri.

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
