>Per calcolare l'[[068 Integrali Indefiniti|integrale indefinito]] di una [[011 Funzioni|funzione]], è possibile *scomporre e ricomporre la funzione* integranda *in una somma di due o più integrali* tramite la [[070 Proprietà degli Integrali#Addizione e Sottrazione di Integrali|proprietà della somma]].

Si utilizza quando il grado dell'incognita è uguale sia al numeratore che al denominatore.

> [!example]+ <font color="orange">Esempio</font>
>Calcolare:
>>$$\int \frac{x}{x+1}\, dx$$
>
>$$\begin{align*} \int \frac{x}{x+1}\, dx &= \int \frac{x+1-1}{x+1}\, dx \\ &= \int \frac{x+1}{x+1}- \frac{1}{x+1}\, dx \\ &= \int 1-\frac{1}{x+1}\, dx \\ &= [\text{proprietà della somma/sottrazione}] \\ &= \int 1\, dx - \int \frac{1}{x+1}\, dx \\ &= x + c - (\ln(|x+1|) + c) \\ &= x + c - \ln(|x+1|) - c \\ &= x - \ln(|x+1|) \end{align*}$$

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
