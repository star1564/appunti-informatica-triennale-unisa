>Sia $n> 0$ un intero non negativo. 
>La **Potenza** di una stringa è il contenuto della stringa *ripetuto* $\textcolor{#61AFEF}n$ *volte*. È quindi la [[031 Concatenazione di una Stringa|concatenazione]] di quella stringa con sé stessa per $n$ volte.
>$$\overbrace{xx\dots x}^k$$



La potenza alla 0 di una stringa è la *[[030 Ricorsione di Stringhe#Stringa vuota|stringa vuota]]*.

> [!example]+ <font color="orange">Esempio</font>
>Se $w=cat$, allora:
>- $w^0=\lambda$,
>- $w^1=cat$,
>- $w^2=catcat$,
>- $w^3=catcatcat$,
>- $...$


> [!quote] Definizione Ricorsiva della Potenza di una Stringa
> Sia $w$ una stringa.
> 
> **Passo Base**: $w^0=\lambda$.
> **Passo Ricorsivo**: $w^{n+1}=w^n\cdot w$, $\forall n\geq0$.

---

> [!note] Parentesi
> È necessario racchiudere tra parentesi la stringa da elevare alla potenza se ha lunghezza maggiore di uno.
> $$(ab)^2 = abab \textcolor{#CC241D}{\neq} abb = ab^2$$
> 
>L'elevamento a potenza ha precedenza rispetto alla concatenazione.

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
