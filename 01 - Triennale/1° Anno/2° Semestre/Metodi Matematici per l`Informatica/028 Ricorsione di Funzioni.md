La *definizione ricorsiva* di una funzione avente come [[016 Variabile (MMI)#^98c8c0|dominio]] l'insieme $\mathbb{N}$ dei numeri interi non negativi (compreso lo $0$) consiste di *due passi*.

> [!quote] Definizione Ricorsiva di una Funzione Matematica
> - **Passo Base**: Specifica il *valore della funzione in zero (oppure uno)*.
> - **Passo Ricorsivo**: Fornisce la *regola* che permette di ottenere il valore della funzione su un intero *attraverso i valori* della funzione su interi *più piccoli*.

Le definizioni ricorsive possono a volte includere una *regola di esclusione*. La regola di esclusione specifica che un insieme ricorsivamente definito contiene *solo gli elementi specificati* dal `PASSO BASE` e dal `PASSO RICORSIVO`.

> [!example]- <font color="orange">Esempio</font>
>Consideriamo la funzione $f:\mathbb{N}\to \mathbb{N}$, definita ricorsivamente come segue: $$f(0)=3,$$$$f(n+1)=2f(n)+3$$
>Per calcolare il valore di $f(5)$ possiamo utilizzare due metodi.
>1. Partire dal passo base ed arrivare man mano all'intero da calcolare. 
>
>![[Pasted image 20220415120137.png|600]]
>
>2. Partire dal numero da calcolare e scendere, mettendo in sospeso il calcolo della funzione attuale, man mano verso il passo base.

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
