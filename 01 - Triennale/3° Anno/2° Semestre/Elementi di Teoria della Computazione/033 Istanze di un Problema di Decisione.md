>L'**Istanza** di un [[001 Logica Proposizionale#^proposizione|Problema di Decisione]] è un *particolare insieme di input* per quel problema.

> [!example]- <font color="orange">Esempio</font>
>I numeri $3,4,6$ fanno parte di 3 separate istanze per il problema decisionale che deve vedere se un numero è primo.
>- $3$ è un'istanza con risposta `Vero`
>- $4,6$ sono due istanze con risposta `Falso`

### Insieme delle istanze

>L'**insieme delle istanze** rappresenta *tutte le istanze* che un problema di decisione ha. 

Questo ci permette di capire *se esiste almeno una soluzione* per un problema.

È formato dall'[[008 Unione tra Insiemi|unione]] di due [[002 Sottoinsiemi|sottoinsiemi]]:
- quello delle *istanze con risposta `Vero`* (**istanze affermative**)
- quello delle *istanze con risposta `Falso`* (**istanze negative**)

> [!example]- <font color="orange">Esempio</font>
> L'insieme delle istanze per il problema dei numeri primi è $\{n\in\mathbb{N}\ |\ n>1\}$, ovvero l'unione dell'insieme dei numeri primi e quello dei numeri non primi o composti.

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
