I consumer e i provider che interagiscono tra loro sul Web devono essere in grado di trovare le informazioni che consentono loro di interconnettersi. O il consumatore conosce la posizione esatta del servizio che vuole invocare o deve trovarlo. 

>**UDDI** fornisce un approccio standard per *individuare le informazioni su un [[053 Web Service|Servizio Web]] e su come invocarlo*. Il fornitore del servizio pubblica un [[054 WSDL|WSDL]] in un registro UDDI disponibile su Internet. In questo modo, il servizio può essere individuato e scaricato dai potenziali utenti. 

È facoltativo, poiché è possibile invocare un servizio Web senza UDDI se si conosce già la posizione del servizio Web attraverso l'[[002 URL|URL]].

UDDI è un *registro di servizi Web basato su [[034 XML|XML]]*, simile a un elenco di Pagine Gialle, in cui le aziende possono registrare i propri servizi. La registrazione include il tipo di azienda, la posizione geografica, il sito web, il numero di telefono e così via. Altre aziende possono quindi cercare nel registro e trovare informazioni su specifici servizi web. Queste informazioni forniscono metadati aggiuntivi sul servizio, descrivendone il comportamento e la posizione effettiva del documento WSDL.

> [!note] UDDI è stato dimesso nel gennaio 2006

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
