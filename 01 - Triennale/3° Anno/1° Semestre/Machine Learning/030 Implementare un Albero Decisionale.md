Il criterio di split da utilizzare per la radice riguarderà la feature "User interest" con il valore di **Fashion**, poiché questa combinazione raggiunge l'impurità di Gini più bassa (e implicamente l'[[029 Metriche di Valutazione#Information Gain (Entropia)|Information Gain]] più elevato).

*Se ci accontentiamo di un albero profondo un solo livello*, possiamo fermarci qui assegnando la giusta l'etichetta 0 del ramo sinistro e l'etichetta 1 del ramo sinistro come classe maggiormente rappresentata.

![[Pasted image 20231216153857.png|600]]

È possibile creare un criterio di split utilizzando varie operazioni, come:
- uguale a, ==
- diverso da, !=
- operatori logici, [[002 AND (MMI)|AND]], [[003 OR (MMI)|OR]], ...

*In alternativa*, si può andare oltre, costruendo il secondo livello a partire dal **ramo sinistro**, dato che il ramo destro non può più essere ulteriormente diviso.

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
