Affrontare lo **Sbilanciamento delle Classi** è una delle sfide comuni nell'ambito dei modelli di machine learning.

Le classi si dicono *Sbilanciate* quando esiste una *grande differenza* nel numero di campioni tra le diverse classi dei dati di addestramento. 

> [!example] <font color="orange">Esempio</font>
>In un set di dati di immagini a raggi X, potremmo avere il 90% di immagini di persone sane e solo il 10% di immagini di persone malate.

Questo problema ha un impatto significativo non solo nei task di [[007 Task di Classificazione|Classificazione]] ma anche in quelli di [[008 Task di Regressione|Regressione]].

### Problematiche
I modelli di Machine Learning lavorano in maniera **ottimale** quando la *distribuzione dei dati è più bilanciata*. Infatti, le classi sbilanciate rendono difficile l'apprendimento per vari motivi.

![[Pasted image 20231215202347.png|700]]

Una prima motivazione è legata al fatto che il nostro modello potrebbe *non avere campioni sufficienti per riuscire a classificare* le classi minoritarie. Se, nel caso peggiore, non si hanno istanze di una determinata classe, il modello assumerà che *quella classe non esiste*.

Un'altra problematica è che un modello potrebbe rimanere bloccato in una *soluzione non ottimale*, senza apprendere pattern e strutture presenti nei dati.

Ci sono diversi modi per [[019 Gestire Dataset Sbilanciati|Gestire Dataset Sbilanciati]].

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
