In Java l'[[015 Ereditarietà|Ereditarietà]] viene attuata con la keyword **`extends`** che deve essere usata nella dichiarazione della classe.

```Java
class A extends B {
	// ...
}
```

Questo implica che ogni istanza della classe `A` sarà anche di tipo `B` ed avrà a disposizione tutti i metodi e attributi della classe `B`.

Nella [[005 UML|Notazione UML]] si esprime la relazione di ereditarietà fra `A` e `B` mediante una freccia che va da `A` verso `B`.
