Nella programmazione orientata agli oggetti, l'**Ereditarietà** (o **Derivazione**) è una relazione esistente tra una classe *più generica*, detta **Superclasse**, e una *più specifica*, detta **Sottoclasse**. ^28eda7

La sottoclasse "*eredita*" le [[011 Variabile Esemplare|Variabili di Stato]] ed i [[Metodi]] della sua superclasse. ^a5acea

Ogni istanza di [[Oggetti|Oggetto]] della sottoclasse è utilizzabile in ogni parte del codice della superclasse.

Consideriamo, come esempio, la relazione esistente tra i diversi tipi di veicoli.

![[Pasted image 20221021174140.png|535]]

In questo caso `Car` è una sottoclasse di `Vehicle`, quindi `Vehicle` è la superclasse di `Car`.
Allo stesso modo `Sedan` è una sottoclasse di `Car`, quindi `Car` è la superclasse di `Sedan`.

Questo perché le automobili condividono molte caratteristiche con tutti gli altri veicoli, come la capacità di trasportare persone da un luogo a un altro.

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
