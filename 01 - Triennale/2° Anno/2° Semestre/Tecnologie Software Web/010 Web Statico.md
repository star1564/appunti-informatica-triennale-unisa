>Il **Web Statico** è un tipo di sito web in cui il contenuto viene *generato una volta* e poi *rimane invariato* fino a quando non viene aggiornato manualmente dal proprietario del sito. 

In un sito web statico, tutte le pagine sono predefinite e non ci sono funzionalità avanzate come la *personalizzazione dinamica dei contenuti* in base all'utente o la possibilità di effettuare *ricerche* interne sul sito. Inoltre, i siti web statici non richiedono l'utilizzo di una [[001 Introduzione a Basi di Dati#^f0d93b|base di dati]] o di altri strumenti di elaborazione server-side per funzionare.

Sono *poco flessibili e scalabili*, poiché ogni pagina deve essere creata manualmente e ogni modifica richiede un nuovo caricamento del sito web.

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
