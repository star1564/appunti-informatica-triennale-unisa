Un **Sito Web Responsive** è un sito che si *adatta* e si visualizza in modo ottimale su *diversi dispositivi*, come computer desktop, tablet e smartphone. Questo significa che il layout e il contenuto del sito si ridimensionano e *si riorganizzano automaticamente* per adattarsi alle dimensioni dello schermo del dispositivo in uso.

Ci sono 3 tipi differenti di layout.

---
### Layout Fluido
>Il **Layout Fluido** si basa sul sistema relativo dei *valori con percentuali*, invece di avere valori con pixel.

![[Pasted image 20230615110429.png]]

---
### Layout Adattivo
>Il **Layout Adattivo** si basa sull'utilizzo di *valori con dimensioni fissate in pixel* ma adottiamo queste dimensioni in base alla dimensione del browser o [[026 Grandezza Dispositivi#Impostare il Viewport|Viewport]].
>Utilizzammo per questo tipo di layout le [[026 Grandezza Dispositivi#Media Query|Media Query]].

![[Pasted image 20230615111032.png]]

Il problema principale è quello di scegliere i breakpoint dove inserire le media query.

---
### Layout Responsive
>Il **Layout Responsive** è un mix tra il layout fluido e adattivo: utilizza le *unità in percentuali* e i *breakpoint*.

![[Pasted image 20230615111306.png|500]]

È estremamente utile quando si riduce al volo la dimensione del browser/viewport. Infatti, *il layout varia dinamicamente* (responsive) in base alla dimensione attuale.

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
