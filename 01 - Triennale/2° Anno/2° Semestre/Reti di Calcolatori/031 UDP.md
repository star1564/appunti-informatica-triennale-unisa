L'**UDP** (*User Datagram Protocol*) è un protocollo di comunicazione nelle reti. 

>A differenza del [[030 TCP|TCP]], l'UDP è un protocollo riguardante la connessione **Connectionless**, rendendolo *poco affidabile* ma *più veloce*.

Prima dell'invio dei dati, *non c'è alcun tentativo* di stabilire una connessione o di verificare se i pacchetti sono arrivati correttamente (non viene eseguito il processo di [[034 Apertura e Chiusura connessione TCP|Handshake]]).

Infatti, i pacchetti possono essere perduti durante il tragitto o addirittura consegnati fuori sequenza all'applicazione.

![[Pasted image 20230909121703.png|300]]

Inoltre, UDP cerca di maneggiare i dati da inviare *in un unico pacchetto*, in modo tale che in ricezione si riceva il flusso completo di dati dell'applicazione ([[030 TCP#^ef1e62|in opposizione al TCP]]). 

> [!success] Vantaggi
> Nonostante sia inaffidabile, UDP offre alcuni vantaggi rispetto a TCP: 
> - Può utilizzare trasmissione multicast o [[002 Rappresentazione di Reti#Broadcast|broadcast]]; 
> - È molto più leggero ed efficiente grazie alla mancanza di controlli del pacchetto.

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number
  const pattern = new RegExp(`^${targetNumber}`);
  return pattern.test(inputString);
}

function sort2Array(array){
	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])
	else if (parseInt(extractNumberFromString(array[0])) > parseInt(extractNumberFromString(array[1])))
		res.push(array[1], array[0])
	else
		res.push(array[0], array[1])
	
	return res
}
