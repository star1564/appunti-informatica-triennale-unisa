>Sia $X$ una [[034 Variabili Aleatorie Continue|variabile aleatoria continua]] con valore atteso $\color{#61AFEF}\mu=E[X]$, la **Varianza** di $X$ è così definita 
>$$\color{#CC241D}\text{Var}(X) = E[(X-\mu)^2]=\int_{-\infty}^{\infty} (x-\mu)^2 \cdot f(x) dx$$con $f(x)\geq 0$ per ogni $x$.

> [!quote] Formula Alternativa
>$$\textcolor{#CC241D}{\text{Var}(X)=E[X^2]-\mu^2}=E[X^2]-[E(X)]^2=\int_{-\infty}^{\infty} x^2 \cdot f(x) dx - \left(\int_{-\infty}^{\infty} x \cdot f(x) dx\right)^2$$

### Proprietà della Linearità
Anche la varianza gode della [[035 Valore Atteso di una VA Continua#Proprietà di Linearità|proprietà di linearità]]. 

>Sia $X$ una variabile aleatoria continua. Se $a$ e $b$ sono costanti reali, allora $$\color{#CC241D}\text{Var}(a\cdot X+b)=a^2\cdot \text{Var}(X)$$

function extractNumberFromString(inputString) {
	const match = inputString.match(/^\d{3}/);
	
	if (match) {
		return match[0];
	} else {
		return null; // Return null if no match is found
	}
}

function startsWithNumber(inputString, targetNumber) {
  // Use a regular expression to check if the string starts with the target number

}

	let res = []
	
	if (array[0] == undefined)
		res.push(array[1], array[1])
	else if (array[1] == undefined)
		res.push(array[0], array[0])

	else
		res.push(array[0], array[1])

}
